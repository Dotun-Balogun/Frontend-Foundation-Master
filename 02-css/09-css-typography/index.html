1. Font Families

Serif Font (Playfair Display)

Serif fonts have small lines or strokes attached to the ends of letters. They're often used for traditional, elegant, or formal designs.

Sans-Serif Font (Inter)

Sans-serif fonts are clean and modern without decorative strokes. They're highly readable on screens and popular for web design.

Monospace Font (Fira Code)

Monospace fonts have equal spacing for each character. Perfect for code, tables, and technical content.

4. Spacing & Readability

Line Height (Leading)

Tight line height (1.2) - This can feel cramped and is harder to read, especially for longer paragraphs. Use sparingly for headings or short text.

Normal line height (1.5) - This is ideal for body text. It provides good readability and is recommended for most content.

Loose line height (2.0) - This creates more breathing room but can feel disconnected. Good for poetry or emphasis.

Readable Line Length

The ideal line length for readability is 50-75 characters (about 8-12 words). This paragraph demonstrates proper measure for comfortable reading without eye strain.

Lines that are too long force readers to work harder, moving their eyes across excessive horizontal distance, which can cause fatigue and reduced comprehension, especially for people with dyslexia or other reading difficulties.
